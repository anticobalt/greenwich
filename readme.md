# Greenwich

![Preview](screenshots/preview.jpg)

## Features
- Conversion of dates/times to other time zones
- Lists all time zones and their corresponding UTC offsets
- Preloaded and custom themes
- Swipe to delete
- Drag and drop to re-order 

## Acknowledgements
